CryptoChill <> Onramper

CryptoChill Onramper integration example

IFRAME SDK modal

CryptoChill provides an easy and automated way to integrate Onramper.com widget. Once you have created Onramper.com API Key and got it approved, you need to add the Onramper API Key to your preferred Configuration Profile.

Once you're done now you can now use CryptoChill <IFRAME> widget using following src attribute: https://cryptochill.com/sdk/v1/onramper/YOUR-PROFILE-ID-HERE/.

Code:
Optional GET Parameters:
  • passthrough – specify passthrough data for the resulting invoice. For JSON object use urlencoded value.
  • invoice_id – specify existing invoice_id to initiate iframe for existing invoice address. In such case currency selector will be hidden and invoice currency will be automatically selected.
  • For example: 
    <iframe src="https://cryptochill.com/sdk/v1/onramper/YOUR-PROFILE-ID-HERE/?passthrough=1234" ...
