CryptoChill <> WordPress Woocommerce

CryptoChill WordPress Woocommerce cryptocurrency payment plugin integration example.

CryptoChill WordPress/WooCommerce plugin provides an easy and automated way to integrate CryptoChill gateway to Woocommerce checkout module. Follow the instructions under

  1. 1. Download, install and activate plugin.
  2. 2. Enable payment method under WooCommerce > Settings > Payment Methods.
  3. 3. Click Manage and follow instructions.
